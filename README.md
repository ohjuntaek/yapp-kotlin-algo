# yapp-kotlin-algo

## 규칙

- 스터디 시간 전 문제를 정해서 readme.md 파일에 올리고 프로젝트를 생성해 푸시 하겠습니다.  
- 인텔리제이 프로젝트 파일이 생성되면, src안에 .kt 파일을 생성해서 푸시해주시면 됩니다.

```text
ex) "기능개발" 문제, 1회차, 1번 문제, 깃헙아이디 ohjuntaek  
=> 1/1_1_기능개발_ohjuntaek.kt
```
